# 🚧 Road Construction Monitoring Dashboard

A modern Flutter application for monitoring and managing road construction projects across multiple districts and government authorities. The application provides an intuitive dashboard with project analytics, authority performance tracking, project comparison, notifications, and detailed project information using the BLoC architecture and Repository Pattern.

---

## ✨ Features

### 🏠 Dashboard
- Project summary cards
- Total Budget & Road Length
- Overall Progress
- Recent Projects
- Interactive navigation

### 📂 Project Management
- View all projects
- Search projects
- Filter by District
- Filter by Authority
- Filter by Status
- Sort projects
- Detailed project information

### 📊 Analytics
- Project Progress Bar Chart
- Monthly Completion Line Chart
- Project Status Pie Chart
- Interactive status cards
- Project statistics

### 🔄 Compare Projects
Compare any two projects based on:
- Budget
- Road Length
- Progress
- Authority
- Contractor
- Status
- Timeline

### 🏛 Authority Performance
- Authority-wise project statistics
- Average project progress
- Total budget allocation
- Completed project count
- Authority-specific project list

### 🔔 Notifications
- Project completion alerts
- Budget approval updates
- Progress notifications
- Mark as Read
- Mark All as Read
- Swipe to Delete

### 📋 Project Details
- District
- Authority
- Contractor
- Budget
- Road Length
- Start Date
- End Date
- Progress
- Materials Used

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/AKASH7577/road-construction-dashboard.git
```

## Navigate to the Project

```bash
cd road-construction-dashboard
```

## Install Dependencies

```bash
flutter pub get
```

## Run the Application

```bash
flutter run
```

# 🏗 Architecture

This project follows the **BLoC (Business Logic Component)** architecture combined with the **Repository Pattern** to ensure clean, scalable, and maintainable code.

**Architecture Flow**

```
UI
 │
 ▼
BLoC
 │
 ▼
Repository
 │
 ▼
Local JSON Data
```
